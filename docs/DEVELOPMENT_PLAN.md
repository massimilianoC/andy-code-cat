# Development Plan

This document is the stable planning entry point for coding agents and contributors.
It exists to keep older cross-references valid while the detailed roadmap lives under `docs/project/ROADMAP.md`.

---

## Current Status

Andy Code Cat is being developed as an open, self-hostable AI website generation platform built around:

- a Clean Architecture Express API
- a Next.js web application
- MongoDB and Redis services
- export and publishing workflows
- multi-provider LLM support

---

## Current Priorities

### R1 — Prompt Architecture Layer

Current focus areas include:

- modular system prompt composition
- reusable prompt-task infrastructure
- style and moodboard context enrichment
- export and publish reliability
- safer operational governance for public collaboration
